# Discord Nitro Generator
A discord nitro generator made with python requests

##### Requirements:
  1. Python 3.6+
  2. Python Requests
 
##### Usage:
  1. Get HTTPS proxies and put them into proxies.txt
  2. Run Generator.py
  3. Input threads and start.
